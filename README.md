# Meu Portfólio

Portfólio Pessoal de Desenvolvedor Web
Descrição

Este projeto consiste no desenvolvimento de um portfólio pessoal online, criado com o objetivo de apresentar o meu perfil profissional como desenvolvedor web, as minhas competências técnicas, os projetos realizados e os meus contactos.

O portfólio foi desenvolvido com foco numa interface moderna, intuitiva e responsiva, permitindo uma boa experiência de utilização em computadores, tablets e dispositivos móveis.

Objetivos do Projeto
Criar uma presença profissional na web;
Apresentar competências em desenvolvimento Front-End;
Divulgar projetos pessoais e académicos;
Facilitar o contacto com recrutadores, clientes e outros desenvolvedores;
Demonstrar conhecimentos de design e desenvolvimento de interfaces responsivas.
Funcionalidades
Página inicial de apresentação;
Secção "Sobre Mim";
Apresentação das competências técnicas;
Galeria de projetos desenvolvidos;
Ligações para redes profissionais e repositórios;
Formulário ou secção de contacto;
Design totalmente responsivo.

Como Executar o Projeto
Executar Localmente
Faça o download ou clone o repositório:
git clone https://github.com/seu-utilizador/portfolio.git
Abra a pasta do projeto.
Execute o ficheiro index.html no navegador ou utilize a extensão Live Server do Visual Studio Code.
Publicação

O projeto encontra-se publicado na plataforma Vercel, permitindo o acesso online ao portfólio através de um endereço web público.

Competências Demonstradas
